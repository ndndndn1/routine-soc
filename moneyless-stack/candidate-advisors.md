# candidate-advisors.md (run 1, skeleton)

본 파일은 run 1 에서는 방향성 초안만 기록한다. 실제 교수 선정은 최근
2년 publication + lab URL + 협업 가능성 3축 확인 후 다음 run 에서 구체화.

> 주의: 아래 정보는 현 run 에서 crawl 되지 않은 상태다. 이름/소속은
> 레퍼런스 논문에서 직접 추출한 **저자 이름** 이며, 연구실 URL 과
> 협업 가능성은 **아직 검증되지 않았다** (근거 부족).
